# Sobre o projeto

Java Spring Investimentos é uma aplicação back end. A aplicação consiste em uma pesquisa de preferência de investimentos a gosto dos usuarios, onde os dados são coletados no pelo login do usuario, e depois são listados os dados e melhores investimentos.

# Tecnologias utilizadas
## Back end
- Java
- Spring Boot
- JPA / Hibernate
- Maven
- Lombok
## Implantação em produção
- Back end: Heroku
- Banco de dados: Postgresql

# Como executar o projeto

## Back end
Pré-requisitos: Java 11

```bash
# clonar repositório
git clone https://github.com/MunizVitor/Java-Spring---INVESTIMENTOS

# entrar na pasta do projeto back end
cd agregador_de_investimento

# executar o projeto
./mvnw spring-boot:run
```
